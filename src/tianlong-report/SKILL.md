## 🎯 功能描述

基于「天龙引擎」交付标准的工业级自动化报告系统：
- Python (FastAPI) 驱动
- 企业级 Word 执行周报自动编排
- 100% 质量验证

## 🔧 使用方式

```bash
# 基础用法
/tianlong-report <报告类型> <数据源>

# 示例

```

## 📋 输出格式

### 1. 报告类型
- 📊 **执行周报** - 团队周度工作汇报
- 📈 **项目进度报告** - 里程碑追踪
- 📝 **技术文档** - API 文档/设计文档
- 📋 **会议纪要** - 结构化会议记录

### 2. 报告结构
```
┌─────────────────────────────────────────┐
│ 报告标题                                │
├─────────────────────────────────────────┤
│ 1. 本周完成（分类展示）                 │
│ 2. 进行中状态                           │
│ 3. 下周计划                             │
│ 4. 风险/阻塞（如有）                    │
│ 5. 需要支持/决策项                      │
└─────────────────────────────────────────┘
```

### 3. 导出格式
- ✅ Word (.docx) - 正式交付
- ✅ PDF - 打印/分享
- ✅ Markdown - 内部协作
