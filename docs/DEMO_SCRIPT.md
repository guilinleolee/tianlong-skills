## 🛠️ 制作工具

### 方案 A：Terminal Recorder（推荐）
```bash
# 安装
npm install -g terminal-recorder

# 录制
tr record demo.trc

# 导出 GIF
tr export demo.trc --format gif --output demo.gif
```

### 方案 B：ScreenToGif（Windows）
1. 下载：https://www.screentogif.com/
2. 录制终端窗口
3. 添加文字标注
4. 导出 GIF

### 方案 C：LICEcap（跨平台）
1. 下载：https://www.cockos.com/licecap/
2. 框选录制区域
3. 直接录制为 GIF
